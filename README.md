# Ai là triệu phú

## Description
Game giả lập gameshow truyền hình "Ai là triệu phú" với đầy đủ các đặc điểm không khác gì trò chơi thực tế: 15 câu hỏi, 3 sự trợ giúp, đồ họa đẹp, âm thanh mãn nhãn, mô phỏng giống như thật.
